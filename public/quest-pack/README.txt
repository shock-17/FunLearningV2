Quest World Texture Pack (optional)

This folder is OPTIONAL. If you add images here, the Phaser mini-game will use them.
If you don't add anything, the game falls back to generated placeholder textures.

IMPORTANT (copyright / licensing):
- Only use assets you own or that are licensed for commercial use (CC0 / permissive).
- Keep a note of the source + license for anything you include.

Expected files (PNG recommended):
- sky.png            (background, will be tiled; any size)
- platform.png       (platform tile, will be tiled; any size)
- player.png         (player sprite, recommended ~32x48)
- npc_math.png       (NPC sprite, recommended ~44x54)
- npc_english.png
- npc_mandarin.png
- gate.png           (gate sprite, recommended ~26x140)

Tip: if you want a Terraria-like vibe without copyright issues, look for CC0 pixel-art tilesets
on sites like OpenGameArt or Kenney (and confirm the license is CC0/permissive).
